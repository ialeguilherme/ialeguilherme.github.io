Fazer Pedido

Caso de uso primário

Atores

Cliente

Pré-condição:

O usuário necessita ter feito o login no sistema.

Fluxo de Eventos principal

  1. O cliente fornece seus nome e endereço a ser entregue o pedido
  2. Escolhe os meio de pagamento, boleto ou cartão
  3. O cliente submete os dados ao sistema
  4. O sistema verifica, marca o pedido como "pendente" e envias as informações para o banco de dados
  5. Quando o pagamento é confirmado, o pedido é marcado como "confirmado" e o codigo de rastreio é dado ao cliente para que possa acompanhar a entrega.
  6. O caso termina

Pós-condição:

O pedido irá ser adicionado ao sistema.
