Produtos com desconto

Caso de uso primário

Atores

Cliente

Pré-condição:

Protudo estar com desconto

Fluxo de Eventos principal

  1. O sistema procura se o produto está com desconto
  2. O sistema mostra o desconto ao cliente
  3. O sistema calcula o valor do desconto
  4. O sistema atualiza o total
  5. O caso termina

Pós-condição:

Valor do desconto calculado

Estende

1. Ter feito o pedido antes do caso 5.
