Verificar pedido

Caso de uso primário

Atores

Cliente

Pré-condição:

O usuário ter feito o pedido e estar logado no sistema

Fluxo de Eventos principal

  1. O caso começa quando o cliente clica em "meu pedido".
  2. Usa Buscar pedido
  3. O sistema irá mostrar os dados da situação do pedido
  4. O caso termina

Pós-condição:

O pedido não ter situação alterada
