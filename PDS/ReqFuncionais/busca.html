Buscar pedido

Caso de uso primário

Atores

Cliente

Pré-condição: Não é necessária.

Fluxo de Eventos principal

  1. O cliente deverá buscar na aba "meus pedidos"
  2. O cliente ativa a "Busca".
  3. Se tiver fornecido o numero do pedido:
    • O sistema retorna o pedido e o caso de uso acaba
  4. O cliente deve fornecido o nome de usuario
    • o sistema lista todos os pedidos do cliente
    • o cliente seleciona o pedido
    • o sistema retorna o pedido
    • caso de uso termina

Usado por

  1. Verificar pedido

  2. Canselar Pedido
