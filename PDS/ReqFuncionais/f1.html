Cadastrar Usuário

Caso de uso primário

Atores

Cliente

Pré-condição:

Ser maior de idade

Fluxo de Eventos principal

  1. O cliente fornece seus dados de identificação, localização e endereço de e-mail
  2. O cliente submete os dados ao sistema
  3. O sistema verifica se há conflito de dados com outro usuário e envias as informações para o banco de dados
  4. O sistema envia um e-mail de confirmação para o endereço eletronico do usuário
  5. O caso termina

Pós-condição:

O usuário irá ser adicionado ao sistema.
