Login

Caso de uso primário

Atores

Cliente

Pré-condição:

Ser cadastrado no sistema

Fluxo de Eventos principal

  1. O cliente fornece seus dados de login
  2. O sistema verifica se há cadastro da conta solicitada
  3. O sistema libera o acesso caso a conta esteja cadastrada no banco de dados
  4. O caso termina

Pós-condição:

O usuário terá acesso aos detalhes pessoais e permissão para gerenciar seus produtos.
